## Quick Start (3 Steps)

1. Install Azure PowerShell: `Install-Module -Name Az -Force`
2. Connect to Azure: `Connect-AzAccount`
3. Run audit: `.\Azure-DoD-FedRAMP-Audit.ps1 -AllSubscriptions`
